# Beton Pinsk

Одностраничный статический сайт для строительной компании по продаже бетона в Пинске и ближайших районах.

Контент страницы собран на основе файла [docs.docx](docs.docx). Сайт сделан на `Next.js 16` с `App Router` и настроен на статический экспорт через `output: "export"` в [next.config.ts](next.config.ts).

## Что есть в проекте

- landing page в [app/page.tsx](app/page.tsx)
- глобальные стили в [app/globals.css](app/globals.css)
- метаданные страницы в [app/layout.tsx](app/layout.tsx)
- favicon / иконка сайта в [app/icon.svg](app/icon.svg)

## Локальный запуск

Установить зависимости:

```bash
npm install
```

Запустить dev-сервер:

```bash
npm run dev
```

Открыть в браузере:

```text
http://localhost:3000
```

## Сборка

Production build:

```bash
npm run build
```

После сборки Next генерирует статический результат, подходящий для публикации на обычном статическом хостинге.

## Структура контента

Основные блоки страницы:

- hero-блок с оффером и телефоном
- преимущества доставки
- марки и классы бетона
- шаги заказа и доставки
- карточки с ценами
- FAQ и контактный блок

## Технологии

- Next.js 16.2.4
- React 19
- TypeScript
- Tailwind CSS 4
